Typography Controls

Fine-tune your typography settings to enhance readability and aesthetic appeal. Small adjustments can make a big difference in how text is perceived.

Font weight controls the thickness of the letterforms. Light weights (100-300) work well for large text, while medium weights (400-500) are ideal for body text. Bold weights (600-900) create emphasis.

Letter Spacing

Letter spacing (tracking) is the space between characters. Positive values increase spacing, making text feel more open, while negative values tighten text. Display and all-caps text often benefit from increased letter spacing.

Word Spacing

Word spacing controls the space between words. Increased word spacing can improve readability, especially for dense text, but excessive spacing can disrupt the flow of reading.

Line height (leading) is the vertical space between lines of text. Body text typically benefits from line heights of 1.5-1.6 for optimal readability, while headings often work well with tighter line heights (1.1-1.3).

Font Size

2.5rem
1.125rem
Font size affects both readability and visual hierarchy. Body text typically works best between 16-18px (1-1.125rem), while headings should be substantially larger to create clear hierarchy.

Text Transform

Text transforms alter the capitalization of text. Uppercase can create emphasis but can be harder to read in long passages. Sentence case and title case are typically most readable for body text.

Text Alignment

Text alignment affects readability and aesthetic. Left-aligned text (ragged right) is most readable for longer content. Justified text creates clean edges but can create uneven spacing. Center alignment works for short, decorative text.

Typography Tips & Resources

Anatomy of Typography

Understanding typographic terminology helps in making better design decisions:

  • Baseline: The invisible line where characters sit
  • X-height: Height of lowercase letters
  • Cap height: Height of capital letters
  • Ascenders: Parts of letters that extend above the x-height
  • Descenders: Parts of letters that extend below the baseline
  • Kerning: Adjustment of space between specific character pairs

Font Classification

Fonts generally fall into these categories:

  • Serif: Have small lines at the ends of characters (e.g., Times New Roman). Good for traditional, authoritative content.
  • Sans-serif: No decorative lines (e.g., Helvetica). Clean, modern appearance.
  • Display: Decorative fonts designed for headlines and short text. Not suitable for body text.
  • Monospace: Each character occupies the same width. Used for code and technical content.
  • Script: Mimics handwriting or calligraphy. Best for decorative purposes.

Typography Best Practices

  • Limit your design to 2-3 font families for consistency
  • Create clear contrast between heading and body text
  • Ensure sufficient line height for comfortable reading
  • Maintain line lengths of 45-75 characters for optimal readability
  • Use proper hierarchy to guide the reader's attention
  • Consider the emotional character of fonts when selecting them
  • Test typography at different screen sizes

Accessibility Considerations

  • Maintain sufficient contrast between text and background
  • Use relative units (em, rem) instead of absolute units (px)
  • Avoid using very thin font weights for body text
  • Ensure line height is at least 1.5 for body text
  • Body text should be at least 16px (1rem)
  • Avoid justified text which can create uneven word spacing
  • Don't use all caps for long sections of text

Typography is what language looks like.

Good typography establishes a visual hierarchy for meaning by contrasting styles and sizes. A great font combination creates harmony between elements while ensuring readability and appropriate emphasis.
